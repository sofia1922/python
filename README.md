# reverse_words_package

## Опис
`reverse_words_package` — Python-пакет для реверсування слів у рядку, зберігаючи небуквені символи на їхніх місцях.  
Приклад:  
Вхід: `"a1bcd efg!h"`  
Вихід: `"d1cba hgf!e"`
